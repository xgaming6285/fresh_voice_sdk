# ============================================
# Platform-Specific Notes
# ============================================

# Windows Users:
# - ffmpeg is required for MP3 support
#   Download from: https://ffmpeg.org/download.html
#   Or install via chocolatey: choco install ffmpeg
#
# - PyAudio wheels for Windows:
#   If pip install fails, download wheel from:
#   https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyaudio

# Linux Users:
# - Install system dependencies first:
#   sudo apt-get install python3-pyaudio portaudio19-dev ffmpeg
#   sudo apt-get install libsndfile1

# ============================================
# Installation Instructions
# ============================================

# 1. Create virtual environment:
#    python -m venv venv
#
# 2. Activate virtual environment:
#    Windows: venv\Scripts\activate
#    Linux/Mac: source venv/bin/activate
#
# 3. Upgrade pip:
#    pip install --upgrade pip
#
# 4. Install dependencies:
#    pip install -r requirements.txt
#
# 5. (Optional) Install transcription support:
#    pip install faster-whisper
#    OR
#    pip install openai
#
# 6. Configure environment variables:
#    Copy .env.example to .env and fill in your API keys
#
# 7. Initialize database:
#    python init_superadmin.py
#
# 8. Run the application:
#    python windows_voice_agent.py
